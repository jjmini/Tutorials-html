B-1 集合

1.1 集合运算规律

集合A和B的交集 \[A \cap B = \left \{ x:x \in A且x \in B \right \}\]
集合A和B的并集 \[A \cup B = \left \{ x:x \in A且x \in B \right \}\]
集合A和B的差集 \[A - B = \left \{ x:x \in A且x \in B \right \}\]

他们服从如下定律：

空集率

\[A \cap \varnothing = \varnothing\]

\[A \cup \varnothing = \varnothing\]

幂等率

\[A \cap A = A\]

\[A \cup A = A\]

交换律

\[A \cap B = B \cap A\]

\[A \cup B = B \cup A\]

分配率

\[A \cap \left ( B \cap C \right ) = \left ( A \cap B \right ) \cap C\]

\[A \cup \left ( B \cup C \right ) = \left ( A \cup B \right ) \cup C\]

分配率

\[A \cap \left ( B \cup C \right ) = \left ( A \cap B \right ) \cup \left ( A \cap C \right )\]

\[A \cup \left ( B \cap C \right ) = \left ( A \cup B \right ) \cap \left ( A \cup C \right )\]

吸收率

\[A \cap \left ( A \cup B \right ) = A\]

\[A \cup \left ( A \cap B \right ) = A\]

德$\cdot$摩根率

\[A - \left ( A \cap B \right ) = \left ( A - B \right ) \cup \left ( A - C \right )\]

\[A - \left ( A \cup B \right ) = \left ( A - B \right ) \cap \left ( A - C \right )\]
